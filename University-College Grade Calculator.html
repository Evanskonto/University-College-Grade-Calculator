Help?

This appliation helps you to calculate the average grade across modules and your degree. Knowing the grade average can help you to improve your degree and to get better marks. Degree categories are determined differently at each University-College. As a result, University-College Grade Calculator has been created as a tool for quicly calculating weighted averages wihtout needing to know a lot of things about marking procedures.

How this application works.

This calculator can calculate up to 6 modules. On each box, you should put marks between 0 and 100. Only percenatges between 0 and 100 are acceptable .All the boxes must have marks between 0-100 else you will see a message "Please enter marks between 0-100". When all the boxes have grades inside, you can click the button calculate to see the results.
